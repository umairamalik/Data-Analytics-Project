Customer Behavior Data Analysis Project

This repository presents an end-to-end data analytics project focused on analyzing customer shopping behavior and translating raw transactional data into actionable business insights. The project follows industry-standard practices used by data analysts in real-world business environments and demonstrates the complete analytics lifecycle.

The objective is to showcase practical skills in Python, SQL, and Power BI while emphasizing structured problem-solving, data-driven decision-making, and clear communication of insights.

Project Objectives

Perform data cleaning, transformation, and exploratory analysis on customer behavior data

Design a relational data model and execute analytical SQL queries

Identify customer segments, purchasing patterns, and loyalty indicators

Build an interactive Power BI dashboard for stakeholder analysis

Summarize insights and provide business recommendations

Tools and Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (MySQL / PostgreSQL / MS SQL Server)

Power BI

Jupyter Notebook

Git & GitHub

Project Workflow

Data ingestion and exploratory data analysis in Python

Data cleaning, validation, and transformation

Loading processed data into a SQL database

Writing SQL queries to answer business questions

Connecting SQL to Power BI for visualization

Building an interactive dashboard and extracting insights

Creating a summary report and presentation
